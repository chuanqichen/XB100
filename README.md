# XB100

## install git
```
sudo apt install git
```

## get this repository the first time
```
start from home directory: 
cd 
git clone https://github.com/chuanqichen/XB100.git
```

## get latest update in subsequent labs 
```
always start from XB100 root folder: 
cd
cd XB100
git pull
```
